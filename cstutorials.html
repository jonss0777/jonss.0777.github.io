Ok, that explains how you can create a variable in python. It doesn't explain what it represents.

A variable can be though of as a name for a location of memory in our computer. Visually it looks like this...

Notice how our name("Jack") is mapped toward a block of memory. The physical memory itself already has an id 0x02 and "Jack" is just a name we use to reference it.
